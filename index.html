    Projects

    EEG-Controlled Prosthetic

      Developing prosthetic controlled by brainwaves. The model distinguishes difference in brainwaves varying from different types of motor imagery and determine which action the prosthetic needs to take.

      Team project for Engineering Open House at the University of Illinois.

    Simple To-Do App "EasyToDo"

      Developing an Android-based Application simple, yet interactive user interface with least, needed functionalities.

      For personal use.

    Handwriting Recognition Pen

      Developed paperless pen detecting English Alphabets written using accelerometer and gyroscope.

      Team project for Engineering Open House at the University of Illinois.

    VR Game "Blocks!"

      Player can create blocks and spheres of different size and color to build anything. The object of the game is to enable player to make their imagination into reality with least real-world physics law, gravity.

      The game also contains mini games such as bowling and basketball with the same physical effect as in the real world.

    Stock Prediction Model

      Use historical data of stock price from top tech firms and use regression to build prediction model that analyzes trend to help decision in stock market.

    Chatbot

      Python-based chatbot that can analyze user’s questions in linguistic approach.

    Operating System

      Developed a simple operating system that imitates basic functionalities of Linux.

    Python GUI

      Python-based Graphic User Interface (GUI) to analyze raw data from the sensor and visualize the obtained results.

      For research purpose at Rogers Research Group in Urbana, IL
